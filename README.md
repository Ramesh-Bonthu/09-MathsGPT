🤖 Text to Math Problem Solver using LLaMA

An intelligent AI-powered math problem solver built with LangChain, Groq (LLaMA 3), and Streamlit that converts natural language questions into step-by-step mathematical solutions.

🚀 Features
🧠 Understands natural language math problems
🧮 Performs calculations using a custom calculator tool (NumExpr)
🔍 Supports reasoning-based problem solving
🌐 Integrates Wikipedia tool for additional knowledge
💬 Interactive chatbot-style UI using Streamlit
⚡ Fast responses powered by Groq LLaMA model
📊 Step-by-step reasoning with intermediate calculations

🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
LLM: Groq (LLaMA 3.1)
Framework: LangChain
Math Engine: NumExpr
Tools Used:
Calculator Tool
Wikipedia Tool
Custom Reasoning Tool
